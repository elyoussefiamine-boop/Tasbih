## 🌟 À propos

**Dhikr** est une application web minimaliste conçue pour accompagner les musulmans dans leur rappel quotidien d'Allah (الذكر). Elle offre une expérience pure, sans distraction, avec un retour tactile et visuel à chaque étape.

> « Le meilleur des rappels est ce qui se fait en secret. »

## ✨ Fonctionnalités

### 🎯 Fonctionnalités principales
- 🔘 **Grand bouton tactile** (260px) pour un confort optimal
- 📳 **Micro-vibration** à chaque appui (feedback tactile)
- ⚡ **Flash doré plein écran** à chaque palier atteint (33, 66, 99...)
- 🔊 **Son discret** optionnel pour confirmer chaque comptage
- 🎨 **Motif islamique géométrique** en arrière-plan
- 💾 **Sauvegarde automatique** des préférences (langue, son, vibration)

### 🌍 Multilingue
- 🇸🇦 **العربية** (Arabe) - langue par défaut
- 🇫🇷 **Français**
- 🇬🇧 **English**

### 🎮 Modes de comptage
| Mode | Description |
|------|-------------|
| **33 × 3** | Classique (SubhanAllah, Alhamdulillah, Allahu Akbar) |
| **33 × 1** | Un tour de 33 |
| **99** | 99 noms d'Allah |
| **100** | Istighfar 100 fois |
| **Personnalisé** | Votre propre nombre |

### 📱 Feedback sensoriel
| Action | Feedback |
|--------|----------|
| Chaque appui | Micro-vibration (30ms) + son tick (optionnel) |
| Palier atteint (33, 66...) | Flash doré + vibration forte + bip |
| Objectif atteint | Double bip + vibration + message "Terminé" |

### ⚙️ Options
- 🔊 Activer/désactiver le son
- 📳 Activer/désactiver la vibration
- ↺ Réinitialiser le compteur
